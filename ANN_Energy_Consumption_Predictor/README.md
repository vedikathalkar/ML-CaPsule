# ANN-Based Energy Consumption Predictor

## Overview
This project predicts household appliance energy consumption using an Artificial Neural Network (ANN).

## Features
- Data preprocessing
- Feature scaling
- ANN regression model
- Model evaluation
- Visualization of predictions

## Dataset
Appliances Energy Prediction Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Results
The model successfully predicts appliance energy consumption using environmental and temperature-related features.
